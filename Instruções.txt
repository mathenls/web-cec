A dinâmica consiste em acessar o serviço e retornar o contador de vagas para um determinado usuário.

```URL: ec2-35-164-139-210.us-west-2.compute.amazonaws.com

Verbo: GET
````

`http://ec2-35-164-139-210.us-west-2.compute.amazonaws.com/hirers/111222/opportunities`

O retorno da chamada é uma lista em JSON com todas as vagas para o usuário com o ID informado. 
Neste exemplo o resultado esperado do código que vocês irão desenvolver seria 1, pois o usuário possui apenas uma vaga cadastrada no sistema.
Caso uma nova vaga seja adicionada para este usuário, o resultado esperado é 2.

- O código pode ser feito em *qualquer linguagem* de programação. 

- O código deve esperar um parâmetro de entrada que é o ID do usuário
-Será necessário criar uma conta no bitbucket ou no github para submeter o código para avaliação. 

-Códigos copiados serão desclassificados
